Adaptive Picard Iteration

This page contains functions relating to the core adaptive picard iteration functions.

std::vector<std::vector<double>> adaptive_picard_chebyshev(double *r0, double *v0, double t0, double tf, double dt, double deg, double tol, int soln_size, double *Feval, std::vector<double> &Soln, Orbit &orb, EphemerisManager ephem)

This function controls the flow of the APC propagator from orbit segmentation to interpolating the final output vectors.

Parameters

  • r0 – Initial position in ECEF coordinates (km)

  • v0 – Initial velocity in ECEF coordinates (km/s)

  • t0 – Initial time from epoch

  • tf – Final time from epoch

  • dt – Default timestep

  • deg – degree of the gravity model

  • tol – tolerance for the solution

  • soln_size – size of interpolated output

  • Feval – Gravity function evaluations counter

  • Soln – The solution array

  • orb – The orbit object that stores various properties of the orbit

  • ephem – the cached ephemeris object for interpolatinf moon and sun positions.

Returns

std::vector<std::vector<double> > Returns a vector of interpolated solution values

This function iterates through each segment of the predicted orbit and solves the picard integral to the desired degree of precision for each chebyshev node.

Parameters

  • r0 – Initial position vector (km)

  • v0 – Initial velocity vector (km/s)

  • t0 – Iniitla time (s)

  • t_final – Final time (s)

  • deg – Gravity degree

  • tol – Tolerance

  • Period – Orbital period (s)

  • tvec – Start and end times for all segments (s)

  • t_orig – First segment start and end times

  • seg – Number of segments per orbit

  • N – Degree of Chebyshev polynomials

  • M – Number of Chebyshev nodes

  • prep_HS – Hot start switch flag

  • coeff_size – Length of Chebyshev coefficient array

  • soln_size – Size of solution vectors

  • total_seg – Total number of segments

  • P1 – First integrator operator

  • P2 – Second integrator operator

  • T1 – Chebyshev position matrix

  • T2 – Chebyshev velocity matrix

  • A – Least squares operator

  • Ta – Chebyshev acceleration matrix

  • W1 – Segment time scale factor 1

  • W2 – Segment time scale factor 2

  • Feval – Function evaluation counter

  • ALPHA – Output: Position coefficients

  • BETA – Output: Velocity coefficicents

  • segment_times – Array of segment start and end times

  • orb – Orbit object that stores properties of orbit

  • ephem – Ephemeris manager that provides position of Sun and Moon

Returns

std::vector<std::vector<double> >
